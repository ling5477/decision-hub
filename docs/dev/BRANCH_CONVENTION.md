# 分支命名规范（强制）

格式：
<type>/<activePlanId>__<short-desc>

type：
- feat / fix / refactor / perf / docs / chore / test

short-desc：
- 使用 kebab-case
- 8~30 字符
- 表达清楚改动点

示例：
- feat/2026-02-04_M1_run_gate_mockprovider__run-create-facade
- test/2026-02-04_M1_run_gate_mockprovider__golden-runner-verify
- fix/2026-02-06_M2_persistence_ledger__flyway-baseline
